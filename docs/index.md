# Welcome to Secure Sandbox Docs 🛡️

Secure Sandbox is an elite, open-source educational laboratory environment designed to introduce practitioners to network sockets, vulnerability assessments, secure scripting practices, and privilege escalation vectors. 

Through self-contained sandboxed target systems and interactive automation tools, this workspace provides a complete system playground to learn security defense hands-on.

---

## 🎯 High-Impact Learning Path

Our syllabus comprises **10 progressive lab modules**:

1. **Lab 01: Linux Basics** — Navigate standard shells, evaluate SUID files, and adjust directory permission octals safely.
2. **Lab 02: Network Sockets** — Examine loopback adapters, trace network latency, and query socket connections (`ss -tulpn`).
3. **Lab 03: Service Enumeration** — Discover ports and analyze service banner response headers.
4. **Lab 04: Vulnerability Auditing** — Write automated scripts to spot globally writable system files.
5. **Lab 05: Isolated Web Security** — Study SQL Injection mechanics and enforce parameterized prepared statements.
6. **Lab 06: Linux Privilege Escalation** — Audit scheduled root cron jobs and misconfigured SUID flags.
7. **Lab 07: Windows Privilege Escalation** — Analyze registry installers and unquoted system service paths.
8. **Lab 08: Docker Isolation** — Secure container runtimes, inspect networks, and draft rootless Dockerfiles.
9. **Lab 09: Security Report Writing** — Write high-quality professional audit findings with CVSS severity mappings.
10. **Lab 10: Standard CTF Workflow** — Capture flags methodically without utilizing destructive execution loops.

---

## 🚀 Key Architectural Advantages

* **Strict Localhost Bound**: Standard safety protocols mandate routing all traffic strictly through local adapters (`127.0.0.1`), eliminating exposure risks.
* **ShellCheck & PEP 8 Standards**: Automated linter pipelines confirm that scripts and utilities meet high-grade formatting rules.
* **Continuous Integration**: Direct automated tests confirm operational safety on every code iteration.
* **Interactive React UI Console**: Includes a web console featuring terminal simulators, folder inspectors, copyable macro menus, and interactive CVSS threat calculators.
