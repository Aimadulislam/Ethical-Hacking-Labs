# Lab 09: Professional Security Audit Documentation

## Objectives
- Structure findings into concise technical logs.
- Map CVSS severity scores.

## Step-by-Step Instructions
1. Navigate to `/templates/report_template.md`.
2. Format finding details specifying:
   - Vulnerability Description
   - Impact / Exploitability
   - Concrete Remediation Actions
3. Calculate accurate CVSS v3 vectors to define priority levels.

## Learning Outcomes
- Standardized security reporting formats.
- Documented findings with clear, actionable remediation.
