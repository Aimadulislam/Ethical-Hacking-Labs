# Lab 10: Standardized CTF Flow & Workflow Practice

## Objectives
- Learn systematic, structured workflows to capture flags.
- Understand post-exploitation documentation.

## Step-by-Step Instructions
1. Perform local loopback scans using diagnostic port utilities.
2. Read local flag contents located inside safe educational files (e.g., `flag.txt`).
3. Complete the final lab questionnaire inside `report_template.md`.

## Learning Outcomes
